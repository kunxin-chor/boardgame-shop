# uob-ecommerce

## Running the database CLI

Entering the CLI:
```
mariadb -h 127.0.0.1 -u root -p"mariadb"
```

## Start Server
1. After the first time opening this in Gitpod, change the permission of `./mvnw` with `chmod +x ./mvnw`

2. Start the server with `./mwvnw spring-boot:run`

## reststat server
